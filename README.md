# Tell Time 🇬🇧⏰

>As a French guy in London, when people told me the time, I was always lost. Now thanks to this app, I can confirm what I hear and what I should say to tell the time 😄.

A tiny iOS Swift project with SwiftUI.

## Minimum required

Works with Xcode 11.
